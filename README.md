# Kody's Electric LLC - Website Prototype

A fast, responsive, and conversion-optimized static website prototype designed specifically for Kody's Electric LLC. This project is built to be lightweight, easy to maintain, and ready for instant deployment via Netlify.

## 🚀 Features

* **Mobile-First Responsive Design:** Ensures a seamless user experience across all devices, from smartphones to desktop monitors.
* **Serverless Contact Form:** Utilizes Netlify Forms (`data-netlify="true"`) for hassle-free lead generation without the need for a backend database or PHP.
* **Optimized UX/UI:** Designed with a professional blue and yellow color scheme to establish trust, featuring clear calls-to-action (CTAs) and click-to-call functionality.
* **Zero Dependencies:** Built with pure HTML5 and CSS3. No heavy frameworks, ensuring lightning-fast page load times.
* **SEO-Ready:** Semantic HTML structure, meta tags, and accessibility best practices integrated from the start.

## 🛠 Tech Stack

* **HTML5:** Semantic structure and content.
* **CSS3:** Custom styling using CSS variables for easy theming.
* **Google Fonts:** Utilizing 'Inter' for clean, modern typography.
* **Netlify:** Target deployment platform for hosting and form handling.
